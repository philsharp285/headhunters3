# How to Deploy to Cloudflare Pages ## Option 1: Direct Upload (Recommended for Single File) Since Cloudflare Pages doesn't support single file uploads in the traditional way, here's what to do: 1. **Create a folder** on your computer called `headhunters-site` 2. **Download** `index-upload.html` from this project 3. **Rename it** to `index.html` 4. **Put it** inside the `headhunters-site` folder 5. **Zip the folder** (right-click → Compress/Send to → Compressed folder) 6. Go to **Cloudflare Dashboard** → **Pages** → Your project 7. Click **Create a new deployment** 8. Upload the zip file ## Option 2: Use Wrangler CLI If you have npm/node installed: ```bash npx wrangler pages deploy path/to/headhunters-site --project-name=headhunters ``` ## Troubleshooting If the site shows a blank page or 404: - Make sure the file is named exactly `index.html` (lowercase) - Make sure it's in the root of the upload, not in a subfolder - Clear your browser cache - Wait 2-3 minutes for Cloudflare's CDN to update
